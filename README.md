ShadowByte
ShadowByte is a dark, neon-charged front-end gaming website that showcases games, esports culture, and community through an immersive, cinematic digital experience. The project was built to strengthen my understanding of HTML, CSS, and JavaScript by creating a fully interactive, responsive, and visually engaging website from scratch.

Features

* Cinematic fullscreen hero section with typing animation and mouse parallax
* Custom neon cursor with glow trail
* Animated particle background
* Sticky glassmorphism navbar that activates on scroll
* Active nav-link highlighting synced to scroll position
* Live search with instant game results
* Category selector (RPG, Racing, Horror, Sports, FPS, Puzzle)
* Featured Games grid with tilt, lift, and glow hover effects
* Genre filter bar
* Trending Games carousel with arrow navigation, autoplay, and pause-on-hover
* Game details modal with screenshots, description, platforms, and system requirements
* Playable trailer modal
* Favorites system powered by Local Storage
* Animated community stats counters
* Latest News section with scroll-in animations
* Newsletter signup with animated feedback
* Scroll progress bar
* Back-to-top button
* Loading screen with animated progress
* Fully responsive across desktop, tablet, and phone

Built With

* HTML5
* CSS3
* JavaScript (Vanilla JavaScript)
* Bootstrap Icons
* Google Fonts

Project Goals
The goal of this project was to:

* Practice semantic HTML structure
* Improve CSS layout and animation skills
* Strengthen JavaScript DOM manipulation and event handling
* Build a fully responsive website without using front-end frameworks such as Bootstrap's CSS
* Create a neon, gaming-inspired user interface with interactive features and smooth user experience
